morsetalk_gui func is a module which will 
create a graphical user interface for the 
project.now it has basic morsetalk features
like:ENCODE & DECODE but can be furthur be used
for other features too.the interface has 
dedicated buttons and entry fields for both
encode and decode functions.when button is clicked 
for a function then the result comes both in 
the python terminal and also via an information box.
the GUI also has a quit button which destroys the 
window and closes the program.
